# AIFFEL 양재 2기 2021.12.27 ~ 2022.06.10   
### 아이펠이란?   
모두의연구소에서 플립 러닝 방식으로 AI를 공부하는 스터디 모임입니다.   
플립 러닝은 정해진 분량의 자료를 학습하고, 함께 토론하며 과제를 해결하는 학습법입니다.   
주입식 교육이 아닌 소통을 하면서 성장할 수 있게 주도적으로 학습하고 공유하고 있습니다.
